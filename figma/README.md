# 空库 · UI 视觉稿（细化版）

基于 [`docs/产品/`](../docs/产品/)「先提升认知、空库自喂、自备 Key」方向生成的桌面端高保真参考图。  
实现时按功能点对照 [`docs/工程/功能/`](../docs/工程/功能/)。  
文件为 PNG（非 `.fig` 源文件），可导入 Figma 作底图描摹。

## 设计基调

| 项 | 约定 |
|----|------|
| 产品名 | **空库** |
| 主色 | 静谧青绿 `#2A6F6A` |
| 背景 | 冷石灰侧栏 + 浅灰白主区，浅色模式 |
| 原则 | 空库起步、只按库内答、多领域并存、Key 自备、可导出 |

## 用户路径

```
首次引导(07) → 模型配置(05) → 空库首页(01)
        → 喂养(02) → 解析进度(08) → 归类确认(03)
        → 知识浏览(06) / 条目详情(09) / 对话(04)
        → 拒答规则(11) · 导出备份(10)
```

## 导入 Figma 建议
