# 功能 09 · 知识浏览与条目

> UI：[figma/06-知识浏览](../../../figma/06-知识浏览.png)、[09-条目详情](../../../figma/09-条目详情.png)  
> 产品：[01](../../产品/01-知识库定位.md)、[02](../../产品/02-喂养与归类.md)

## 1. 功能目标

按分类浏览已入库知识；搜索标题/摘要；看条目详情（摘要、原文切片、相关）；支持改分类、删除、跳转对话。

## 2. 技术要点

| 项 | 做法 |
|----|------|
| 列表 | `GET /api/entries?category=&q=&page=` |
| 分类计数 | `GET /api/categories` 带 `count`；0 条分类显示空态引导喂养 |
| 详情 | `GET /api/entries/{id}` + chunks 预览 |
| 更新 | `PATCH` 标题/摘要/标签；`DELETE` 软删或硬删（硬删同步向量） |
| 多领域 | 分类是标签而非互斥库；过滤用 AND/OR 简单即可（MVP：单分类过滤） |

## 3. 实现步骤

1. 浏览页：左表右详情（对齐 figma）
2. 搜索防抖 300ms
3. 详情 Tab：摘要 / 原文切片 / 相关（相关可用同 source 或向量近似）
4. 「在对话中提问」跳转对话页并带 `entry_id` 或预填范围

## 4. 验收

- [ ] 两领域数据可同时存在并分别过滤
- [ ] 空分类有「去喂养」提示
- [ ] 删除后对话不再引用该条目切片
- [ ] 改标签立即反映在过滤器计数
