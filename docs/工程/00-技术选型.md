# 00 · 技术选型

本文敲定 **空库** 自用技术栈。当前个人版：**Electron + SQLite**，不依赖 Docker；用户自备 Key、视频可自动提取文案。  
（历史 NAS + Compose 交付方案见 `archive/docker/` 与产品扩展文档。）

## 1. 总栈（个人版定稿）

| 层 | 选型 | 备选 | 为何定这个 |
|----|------|------|------------|
| 桌面壳 | **Electron** | 仅网页 | 一键起 API；发安装包 |
| 前端 | **React + Vite + TypeScript** | Next.js | SPA 足够 |
| 后端 | **Python FastAPI** + **Knife4j UI**（`/doc.html`） | Node NestJS | 文档解析、ASR、向量生态更熟 |
| 任务队列 | MVP：**BackgroundTasks + 表状态** | ARQ/Redis | 个人机少依赖 |
| 数据库 | **SQLite**（默认 `data/kongku.db`） | 自备 Postgres | 免 Docker；个人够用 |
| 对象/文件 | **本地** `./data` | — | 自用无必要上对象存储 |
| LLM | **OpenAI 兼容 HTTP**（DeepSeek 等） | — | 用户自备 Key |
| Embedding | 同一兼容接口 | 本地小模型 | 优先在线 API |
| 电子书解析 | **pypdf** + **pymupdf** + **RapidOCR** + **ebooklib** | unstructured | 文字层优先；扫描 PDF 可 OCR |
| 视频文案 | **优先字幕**（yt-dlp）→ 再 **ASR API** | 本地 Whisper | 不强制贴文稿 |

## 2. 明确不做（选型边界）

| 不做 | 原因 |
|------|------|
| 默认依赖 Docker / Compose | 个人机简化；Compose 已归档 |
| 在本机跑 7B+ LLM | 内存与稳定性不划算 |
| 绑定单一云厂商 SDK | OpenAI 兼容层可换 |
| 预置通识向量库 | 违反空库原则 |
| 默认公网暴露 | 本机自用 |

## 3. 推荐仓库结构

```
kongku/
├── .env.example
├── README-运行.md
├── data/                        # kongku.db、uploads…
├── archive/docker/              # 历史 Compose（默认不用）
├── apps/
│   ├── web/                     # React + Vite
│   ├── api/                     # FastAPI
│   └── desktop/                 # Electron
├── data/                        # gitignore：kongku.db、上传、导出临时包
└── docs/
```

## 4. 核心依赖（API 侧示意）

```text
fastapi, uvicorn
sqlalchemy / sqlmodel, aiosqlite
（可选）asyncpg — 仅自备 Postgres 时
httpx, pydantic-settings
pypdf, ebooklib, python-multipart
yt-dlp（字幕）, 可选 openai 官方 SDK（兼容 base_url）
```

前端示意：`react`, `react-router`, **Ant Design**（反馈/表单组件；主色对齐 `#2a6f6a`）, `zustand` 或上下文, `tanstack-query`。

## 5. 数据对象（最小表）

| 表 | 作用 |
|----|------|
| `ai_settings` | Base URL、Key（加密或仅存本机）、模型名 |
| `sources` | 一次喂养任务：文件/URL、状态、进度 |
| `entries` | 知识条目（标题、摘要、所属 source） |
| `categories` / `entry_categories` | 多对多分类 |
| `chunks` | 切片正文 + embedding 向量 |
| `settings_refusal` | 拒答开关与话术 |
| `export_jobs` | 可选：导出任务记录 |

## 6. 与产品能力的映射

| 产品能力 | 技术落点 |
|----------|----------|
| 空库 | 无 seed 知识；仅空表 + 拒答默认配置 |
| 自备 Key | `ai_settings` + OpenAI 兼容客户端 |
| 电子书/笔记喂养 | 解析器 → 切片 → embed → 摘要分类 LLM |
| 视频自动文案 | yt-dlp 字幕 → 否则 ASR API |
| 只按库内答 | 检索 top-k chunks；不足则拒答提示词 |
| 可导出 | 打包 `data/` 原文 + entries JSON/MD，排除 Key |
