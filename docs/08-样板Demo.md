# 08 · 样板 Demo

## 1. 目标

让客户相信：**这包在 Docker 下能跑，且会拒答**。  
不依赖你家 NAS 公网 24h 托管。

## 2. 推荐演示方式（由易到难）

1. **录屏**：本机/NAS `compose up` → 打开网页 → 能答 + 拒答各一次  
2. **截图 + 验收题列表** 放闲鱼详情  
3. （可选）视频会议共享屏幕  
4. （可选）短期穿透公网——注意安全，用完关掉  

## 3. 样板包

可做一个 **脱敏垂直样板 Compose**（假店客服或假课程），体验档可卖或免费换询盘。  
与标准档「按客户资料定制」分开。

## 4. 结尾 CTA

> 需要按你的资料定制同款 Docker 包，私信：环境（电脑/NAS/云）+ 场景 + 资料量。
