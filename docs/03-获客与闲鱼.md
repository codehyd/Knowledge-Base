# 03 · 获客与闲鱼

## 1. 你卖的关键词

主打：**Docker 知识库 / 可自托管 / 场景话术或课程 / 会拒答 / 教运行**  
弱化：我给你云账号长期用、扣子平替、装龙虾。

## 2. 标题方向示例

- `店铺客服知识库 Docker 包 自己电脑可运行 只按话术答`
- `课程答疑库 docker-compose 交付 含拒答配置 教你启动`
- `垂直知识库镜像打包 资料可导出 不绑我的服务器`

## 3. 详情页要点

1. 交付什么：Compose/镜像说明、网页端、拒答、（标准档）已整理资料  
2. 跑在哪：客户自己的电脑 / NAS / 云（客户的）  
3. 和在线 AI 差别：边界清晰、资料自己掌握  
4. 环境要求：已装 Docker（或愿意装）；内存建议（如 ≥8G）  
5. 套餐价与「帮你装到你的云」加价  
6. 演示：录屏 / 截图；不承诺你家 NAS 公网 24h 给所有人试用  

## 4. 询盘三问

1. 有没有可用的 Docker 环境（电脑/NAS/云）？  
2. 客服库还是课程库？资料大概多少？  
3. 要「只交包装自己跑」还是「加钱帮你装到你的机器」？

没有环境又只要你托管 → 按当前策略说明「暂不做我方云托管，可等你有机器再做」。

## 5. 成交后

交包 → 协助跑通一次 → 验收 → 导出备份 → 加微信方便补料（管家档）。
