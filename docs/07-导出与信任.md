# 07 · 导出与信任

客户担心资料锁在你这儿——**能导出是成交助推器**，不是理想主义。

## 1. 副业期怎么做

标准档交付 **必须附备份**。前期可 **手工打包**，不必先写导出按钮。

## 2. 导出什么

| 要 | 不要 |
|----|------|
| 客户原文/原文件 | API Key |
| 你整理后的条目、分类、摘要（若有） | 无必要的向量文件 |
| 简单说明：如何再导入其它工具 | 扣留「只有我能打开」的加密壳 |

向量可不给：换环境用客户 Key **重建索引** 即可。对外说清这点。

## 3. 推荐打包形式

`客户名-知识库备份-日期.zip`：

```
README.txt          # 导出说明、库名、日期
原文或整理/
  01-退货政策.md
  02-运费说明.md
  …
（若有）原件/
  xxx.pdf
```

小库也可再给一份合并的 `全部内容.md` 方便阅读。

## 4. 话术

> 附件是你的资料备份，请自行保存。问答索引可在新环境用你的 API Key 重新生成；我们不会靠扣押资料留人。

## 5. 分享页注意

若给访客试用链接：**默认不允许访客导出全文**，避免被薅库。
