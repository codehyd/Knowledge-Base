# 09 · 扩展：获客与套餐（可选 · 副业）

> 自用未稳、认知闭环未跑通之前不必上架。以下供「真要对外」时参考。

## 1. 关键词（若上架）

主打：**空库知识库 / 自托管 Docker / 自己喂养 / AI 归类 / 会拒答 / 自备 API Key**  
弱化：开箱即有百科、我给你云账号长期用、某平台平替。

## 2. 套餐示意

价格自定：`预估小时 × 时薪 × 1.5～2`。

| 档 | 内容 |
|----|------|
| **体验** | 空库 Compose + 短文如何运行 / 如何喂养；可不含远程协助 |
| **标准（主推）** | 完整 Docker 包 + 拒答预设 + 喂养说明 + 协助跑通一次 + 验收（能喂 / 能答 / 会拒）+ 有限次修改 |
| **代整理（选配）** | 标准 + 按对方提供的合法资料帮做首批归类入库（资料仍属对方） |
| **管家（选配）** | 标准 + 包月答疑 N 次；不含无限代运维、你出云钱 / Token |
| **加价** | 对方已有云 / NAS，你 SSH 装好跑通——仍跑在对方资源上 |

## 3. 商品说明必写

1. 交付为软件包与配置，默认 **空库**；对方自备 Docker 环境。  
2. 模型费用由对方 API Key 承担。  
3. 只按导入 / 喂养资料作答；无资料则拒答。  
4. 素材合法由对方保证。  
5. 协助次数 / 时长写死；不含你家 NAS 长期托管。  
6. 含资料导出备份能力说明。  

## 4. 标题方向示例

- `个人知识库 Docker 空库自托管 自己喂书 AI 归类 只按库内答`  
- `自备 DeepSeek Key 的私有知识库 compose 交付 含拒答`  
- `电子书笔记喂养知识库 可导出 不绑我的服务器`  

## 5. 询盘三问

1. 有没有可用的 Docker 环境？  
2. 打算先喂什么材料（书 / 笔记 / 链接）？有没有自备 API Key？  
3. 只要交包装自己跑，还是加钱帮装到他的机器？  

没有环境又只要你托管 → 说明暂不做我方云托管。

## 6. 耗时目标（熟练后）

| 套餐 | 目标 |
|------|------|
| 体验（空库模板包） | 1～2h |
| 标准（含跑通协助） | 3～6h |
| 代整理首批资料 | 另按资料量计 |
| 加价代部署到对方云 | 另计 1～3h |
