# 空库 · 知识库

个人认知知识库：**默认空库**，自行喂养电子书 / 笔记 / 视频链接（视频可自动提取文案）；  
AI 总结归类；对话只按库内答；API Key（如 DeepSeek）自备。  
先提升认知，副业后置。
