## 文案 A（推荐 · 故事感）

**标题：**
用 Vibecoding 搭了个桌面知识库，求拍砖

**正文：**
最近用 vibecoding 做了一个桌面端项目叫「空库」。

定位很简单：
- 默认是空的，不预装通识百科
- 自己喂电子书 / 笔记 / 视频链接
- 对话只按库内作答，超出范围会拒答
- API Key（DeepSeek 等）自备，存在本机

技术栈大概是 Electron + React + FastAPI + Postgres。
扫描 PDF 也能走 OCR，链接优先自动抽字幕/正文。

发出来不是炫技，是想听听大家：
1. 你觉得这个产品方向靠不靠谱？
2. 你会用它装什么资料？
3. 最想先补哪个功能？（移动端？协作？插件？）
4. 如果开源 / 卖安装包，你更能接受哪种？

评论区随便喷，建议和吐槽都欢迎。
也想聊聊 vibecoding 做完整项目时的坑和后续怎么迭代。

#vibecoding #独立开发 #知识库 #Electron #AI应用 #空库
